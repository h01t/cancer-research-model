# Portfolio Blurb

Currently working on a mammography classification research project built on the CBIS-DDSM dataset. I originally explored whether semi-supervised methods like FixMatch could help in low-label medical imaging, but the project evolved into a stronger supervised pipeline after a corrected fine-tuning baseline clearly outperformed the vanilla SSL setups I tested. The current focus is on pushing that supervised baseline further with better training strategy, model analysis, and reproducible experiment tooling.
